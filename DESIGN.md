# DESIGN.md — 「고요의 경제나루」 디자인 가이드 (초안 v0.1)

> 슬로건: **차근차근 함께** · 시리즈: **경제 판단력 교과서**(1권 돈의 언어 ~ 5권 돈의 인생)
> 목표: 일상 경제 판단력을 갖춘 자립인 양성 · 도착점: 스스로 경제 의사결정.

## 0. 레퍼런스 결정

**채택: `tree-public-squares.com`** (일본 크리에이티브 포트폴리오)

| 가져올 것 (그대로) | 버릴 것 (변형) |
|---|---|
| 크림 종이 베이스(`#efece4` 계열) | 멀티 채도 액센트(블루/라임/틸/퍼플/피치) → **단일 저채도 액센트** |
| 순흑 회피, 오프블랙 텍스트 | 12~15rem 초대형 텍스트의 과시 → **절제된 디스플레이 상한** |
| 명조 세리프 + 산세리프 혼용 | GSAP·패럴랙스·Lottie 풀 모션 → **미세 페이드만** |
| 넉넉한 섹션 여백 | — |

**탈락 사유**
- `supaste.com`: 한정수량("5 spots remaining")·반복 CTA·테크 블루 = 전환 중심 다크패턴. 본 프로젝트의 신중·비선정 원칙과 충돌.
- `mira-inc.jp`: 여성향 코스메틱 핑크/라벤더 톤. 타깃·주제 불일치.

**핵심 매핑**
- "책이 마스터 소스" → **인쇄 조판 감성을 웹으로**(typographic, book-like). tree의 명조+종이 결정을 그대로 계승.
- "문장의 무게와 여운" → 대형 명조 인용 + 넉넉한 여백(여백 = 여운).
- "차분·진중·신중 / 자극적 유튜버의 반대" → 채도·모션·대비를 **의도적으로 낮춤**.

## 1. 디자인 원칙 (모든 결정의 상위 기준)

1. **고요함이 기본값.** 움직임·색·소리가 없는 상태가 정상. 추가는 근거가 있을 때만.
2. **책이 마스터, 웹은 파생.** 도서 조판 → 웹/영상/카드뉴스로 추출. 토큰을 4채널이 공유.
3. **자료는 장식이 아닌 근거.** 모든 이미지·수치·인용에는 출처 컴포넌트가 따라온다.
4. **어머니 테스트.** "60대 투자 신생아 어머니가 일상에서 바로 쓸 수 있는가?" — 디자인·문구 모두 이 질문을 통과해야 한다.
5. **단정하지 않는다.** 양면 제시·합니다체. 시각적으로도 "느낌표·붉은 경고·카운트다운" 같은 단정/압박 장치 금지.

## 2. 컬러 팔레트 (의미별)

> 컨셉: **"새벽 강가의 종이"** — 나루(나루터·물·동틀 녘)의 차분함. 저채도·따뜻한 중성.
> 자극적 순색(순적·네온) 전면 금지. 빨강은 경고로도 쓰지 않음(→ 머드/클레이로 대체).

```css
:root {
  /* 종이(배경) */
  --paper:        #F4F1EA;  /* 메인 배경 — tree #efece4보다 살짝 밝게(장문 가독) */
  --paper-raised: #FBF9F4;  /* 카드·인용 등 떠 있는 면 */
  --paper-sunken: #ECE7DC;  /* 구분 섹션·코드/표 배경 */

  /* 먹(텍스트) — 순흑 금지 */
  --ink:        #23282D;    /* 본문 — 대비 약 12:1 (AAA) */
  --ink-soft:   #4A5158;    /* 보조 텍스트 */
  --ink-faint:  #8A9097;    /* 캡션·메타·플레이스홀더 */
  --line:       #DCD6C8;    /* 보더·구분선 */

  /* 액센트 — 단 하나. 고요한 물빛(머먼 청록) */
  --accent:        #2E6B66; /* 링크·핵심·1차 CTA */
  --accent-strong: #245450; /* hover/active */
  --accent-tint:   #E4EDEB; /* 액센트 배경(핵심정리 박스 등) */

  /* 기능색(전부 저채도, 경고도 톤다운) */
  --note:    #5E6B7A;  /* 용어 정의 — 차분한 청회색 */
  --note-bg: #E9ECF0;
  --caution: #9A6B4B;  /* 양면제시의 '주의/오해' — 머드클레이(붉은 경고 대체) */
  --caution-bg: #F1E8E0;
  --cite:    #6B6256;  /* 근거·출처 — 모래/세피아 */
  --cite-bg: #EFE9DD;
}
```

**왜:** 단일 액센트(물빛 청록)는 "나루"의 상징이자 시선 분산을 막아 진중함을 만든다. 빨강을 경고로도 안 쓰는 것은 "충격·반드시" 류 긴장 유발을 시각에서까지 차단하기 위함. 대비는 60대 가독을 위해 본문 AAA(≈12:1)를 확보하되, 톤 안에서만 높인다.

## 3. 타이포그래피

### 폰트 (전부 웹폰트·무료)
| 용도 | 폰트 | 매핑(tree) | 이유 |
|---|---|---|---|
| 제목·인용·도서 조판 | **본명조(Source Han Serif / Noto Serif KR)** | Shippori Mincho | 책·신뢰·여운 |
| 본문·UI·캡션 | **Pretendard** | Shorai Sans | 가독성(특히 60대), 한글 표준 |
| 수치·표 | Pretendard `font-feature: tnum` | — | 경제 데이터 정렬, 근거 가독 |

### 스케일 (접근성 우선 — base 18px)
```css
--text-caption: 0.875rem;  /* 14px — 메타·출처 */
--text-small:   1rem;      /* 16px */
--text-body:    1.125rem;  /* 18px — 본문 기본(60대 고려, 일반보다 크게) */
--text-lead:    1.25rem;   /* 20px — 도입 문장 */
--text-h3:      1.5rem;    /* 24px */
--text-h2:      2rem;      /* 32px */
--text-h1:      2.75rem;   /* 44px */
--text-display: 3.75rem;   /* 60px — 히어로/대형 인용 상한(tree 15rem 대비 절제) */

--leading-body:  1.85;     /* 한글 장문은 넉넉한 행간 */
--leading-tight: 1.35;     /* 제목 */
--measure: 38em;           /* 본문 1줄 ≈ 한글 25~30자(편안한 측정폭) */
```
- 웨이트: 명조 400/600, 고딕 400/500/700. **900 금지**(외침 느낌).
- **올캡스·과장 자간·느낌표 강조 금지.** 강조는 명조 전환 또는 액센트 색·여백으로.

**왜:** tree의 "명조+산세리프(장인+기술)" 구조를 "명조(책·진중) + Pretendard(쉬움·접근성)"로 번역. base 18px·행간 1.85는 어머니(60대)와 중학생 모두를 위한 가독 하한. 디스플레이를 60px로 묶어 과시 대신 여운을 남긴다.

## 4. 레이아웃

```css
--container: 1120px;  /* 페이지 최대 폭 */
--measure:   38em;    /* 본문 컬럼(≈680px) — 읽기 전용 폭 */
--gap: 8px 기반 스케일: 4 · 8 · 12 · 16 · 24 · 32 · 48 · 64 · 96 · 128
```
- **본문은 단일 컬럼**(`--measure`)로 고정 — 독서 경험 우선. tree의 풀블리드는 히어로·도서 표지 등 "장면"에만 한정 사용.
- 그리드: 12컬럼. 도서/영상/카드뉴스 목록은 모바일 1열 → 태블릿 2열 → 데스크톱 3열.
- 여백을 **콘텐츠로 취급**(여백 = 여운). 섹션 상하 패딩 96~128px.
- 모든 터치 타깃 ≥ 44px(60대 모바일 접근성).

**왜:** supaste의 1200px가 아니라 1120px + 680px 독서 컬럼을 택한 건, 이 사이트의 본질이 "전환"이 아니라 "읽기·학습"이기 때문. 책의 판형 감각을 웹으로 옮긴다.

## 5. 핵심 컴포넌트 패턴

### 공통 컴포넌트
- **네비게이션:** 정적(또는 매우 얕은 그림자) 헤더. 명조 워드마크 「고요의 경제나루」 + 최소 링크(시리즈 · 도서 · 영상 · 소개). hover는 얇은 밑줄만. 고정 시 배경 `--paper` 불투명, 강한 셰도우 금지.
- **버튼:** 절제형. 1차 = 액센트 솔리드(`--accent`), 2차 = 보더만. 그라데이션·강한 셰도우·발광 금지. radius 6px. 레이블은 동사+화살표("차근차근 보기 →", "1권 미리보기 →"). **압박 문구·카운트다운·"지금 바로" 금지.**
- **카드:** `--paper-raised` 면 + 1px `--line` 보더(또는 아주 옅은 셰도우 택1). 이미지는 근거/내용일 때만, 장식 컷 금지. 도서 카드(표지·권차·한 줄 요지) / 영상 카드(썸네일·길이·핵심 한 줄) / 카드뉴스 카드.
- **폼(뉴스레터·문의):** 큰 입력 필드·또렷한 라벨·1열. 다크패턴(선체크·숨은 해지) 금지. 제출 후 차분한 확인 문구.

### 콘텐츠 전용 컴포넌트 (이 프로젝트의 정체성)
- **인용/핵심문장 블록** — 명조 대형(`--text-h2~display`) + 넉넉한 여백. "문장의 무게와 여운"의 주 무대. 따옴표·세로선 등 최소 장치.
- **근거·출처 박스(`--cite`)** — 모든 수치·인용에 부착. `출처 · 기준일 · 원자료 링크`. "자료는 장식이 아닌 근거" 원칙의 강제 장치.
- **양면 제시 컴포넌트** — 좌/우(또는 상/하) 균형 2단. "이렇게 보는 시각 / 다르게 보는 시각". 어느 쪽도 색·크기로 우열을 주지 않음(단정 회피).
- **용어 정의 박스(`--note`)** — 중학생 난이도 풀이. 한 화면 한 개념.
- **실전 적용 체크** — "어머니가 일상에서 쓸 수 있는가?" 체크리스트형. 소비·저축·대출·보험·세금·투자·노후 카테고리 태그.
- **시리즈 진행 표시** — 1권(돈의 언어) → 5권(돈의 인생) 5단 스텝. 현재 위치만 액센트.

**왜:** tree의 카드/FAQ/CTA 패턴을 계승하되, "근거 박스·양면 제시·정의 박스"를 1급 컴포넌트로 승격한 것이 이 프로젝트만의 차별점 — 신중함·비단정을 UI 구조로 못박는다.

## 6. 인터랙션 · 애니메이션

- **기본은 정지.** tree의 GSAP/패럴랙스/Lottie 전면 제거.
- 허용: 스크롤 진입 시 **미세 페이드+8px 상승**(180~250ms, ease-out), 링크/버튼 **색·밑줄 전환**(150ms).
- **금지:** 자동재생, 무한 루프 모션, 시차 패럴랙스, 카운트다운, 흔들림/팝, 큰 호버 확대.
- `prefers-reduced-motion: reduce` 시 모든 전환 제거(어린 학생·고령·전정 민감 사용자 보호).
- 영상 임베드는 클릭 후 재생(자동재생·자동음성 금지).

**왜:** 모션의 절제 자체가 "차분한 유튜버"라는 포지셔닝의 시각적 증거. 고요함은 멈춰 있을 때 전달된다.

## 7. 4채널 일관성 (책=마스터)

| 채널 | 적용 |
|---|---|
| **도서(마스터)** | 본명조 조판·종이 톤·단일 액센트가 원본 기준 |
| **웹사이트** | 도서 토큰을 그대로 CSS 변수화(본 문서) |
| **유튜브** | 썸네일: `--paper` 배경 + 명조 제목 + 단일 액센트. 큰 글씨·붉은 화살표·놀란 표정 썸네일 금지 |
| **인스타 카드뉴스** | 1:1/4:5 템플릿, 동일 토큰. 한 카드 한 메시지, 텍스트 과밀 금지 |

공유 토큰: 색·폰트·여백 스케일은 단일 소스(`tokens`)에서 4채널이 import.

## 8. 가드레일 — 회피 항목의 시각 번역

> 카피 규칙(합니다체·학자명 회피·금지표현 10종 등)은 별도 톤 가이드. 여기서는 **시각 규칙**만.

- ❌ 붉은 경고색·네온·고채도 순색 → ✅ `--caution`(머드클레이) 등 저채도로 대체
- ❌ 느낌표 강조·올캡스·900 웨이트·기울임 외침 → ✅ 명조 전환·여백·액센트 색으로 강조
- ❌ 한정수량·카운트다운·"지금/반드시" CTA(=supaste 패턴) → ✅ 차분한 동사+화살표
- ❌ 자극적 구도·과장 합성·사람 표정 클로즈업 썸네일 → ✅ 정물·도식·종이 질감, 근거 이미지
- ❌ 수익률/종목/시점 암시 그래프(상승 화살표·불꽃) → ✅ 중립 도식, 양면 제시, 출처 명기
- ❌ 정치·종교·젠더·지역 상징색/도상 → ✅ 일절 사용 안 함
- ✅ 접근성: 본문 대비 AAA(≈12:1), 액센트 텍스트 AA 이상, 터치타깃 ≥44px, 모션 축소 존중
